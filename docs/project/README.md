# Project Docs Entry Point

Use this document as a quick onboarding reference.

## Quick Start

### Prerequisites
- Node.js 18+
- npm

### Run Locally

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

## Build and Checks

```bash
npm run build
npm run lint
```

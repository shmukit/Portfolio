# Scripts

This folder contains utility scripts for development and debugging.

## 📁 Files

### `debug-db.js`
- **Purpose**: Database debugging script
- **Usage**: Run with `node debug-db.js` to debug database connections and queries
- **Note**: Useful for troubleshooting Supabase connection issues

## 🔧 Usage

```bash
# Run database debug script
node scripts/debug-db.js
```

## 📝 Notes

- These scripts are for development and debugging purposes
- Make sure your `.env.local` file is properly configured before running
- Some scripts may require specific Node.js versions or dependencies
